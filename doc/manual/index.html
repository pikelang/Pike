The 無PC Programming Language

by Lars Aronsson and Fredrik Hbinette

What is this?

無PC (Micro L.P.C., sometimes written ulpc or uLPC) is an interpreted, object-oriented programming language for flexible and yet efficient application development and prototyping. It features multiple inheritance, data abstraction, advanced built-in datatypes, such as associative arrays, dynamic arrays and multi-sets, and high-level character string and array operations.
